# ScrollReaderApp

![ScrollReader Logo](public\icons\ScrollReader144.png)

Anwendung zum Lesen von Bibelversen mittels scrollen wie bei YouTube Shorts.

## Kapitelauswahl

Bibelbuch und Kapitel können gewählt werden. Danach werden sie in einem Cookie gespeichert.
Dort gehts dann weiter für den Nutzer, wenn er die Seite wieder öffnet.
